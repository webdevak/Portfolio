Game Show App

I programmed a browser version of “Wheel of Success”, a word guessing game where players click letters from an onscreen keyboard to try to guess a random phrase. Used JavaScript to create an array of phrases and write functions to choose a random phrase from that array, then split the phrase into letters, then I put those letters onto the game board. Each time the player guesses a letter, I compared the letter the player chose with the random phrase. If the letter is in the phrase, the board updates the chosen letters. A player can keep choosing letters until they make five incorrect guesses. If the letter then chose isn’t in the phrase, one of the player’s 5 guesses is removed. If the player completes the phrase before they run out of guesses, a winning screen will display. If the player guesses incorrectly 5 times, a losing screen will display. A player can guess a letter only once. After they’ve guessed a letter, the program disables that letter.
